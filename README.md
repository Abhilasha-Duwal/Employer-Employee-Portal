# InnovateTech-Assignment

# For node-app
```
cd node-app
npm install
```

## Build the services
### it may take few time when building services intially as it pull images from docker hub.
```
make build
```

## Start local services
```
make start
```

## Stop local services
```
make stop
```

# For react-app
```
cd react-app
npm install
```

## Start react-app
```
npm start
```
